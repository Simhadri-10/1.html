Description list

  • HTML:

  • HTML (HyperText Markup Language) is the standard markup language used to structure and present content on the web. It defines the meaning and structure of web pages by using elements enclosed in tags. These tags help browsers interpret and display content such as text, images, videos, and links.

  • p:

  • To add a paragraph in HTML, you use the p tag. This tag defines a block of text as a paragraph, which browsers automatically format with some space before and after

  • b:

  • In HTML, the b tag is used to make text bold purely for visual styling without adding any extra semantic importance.

  • br:

  • In HTML, the
    tag is used to insert a line break in text, forcing the content after it to appear on a new line
